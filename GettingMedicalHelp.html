Getting Medical Help

    If you think it's an emergency:

  • Call 999

    • If it's not an emergency, but still need help:

  • Talk to a pharmacist for any advice on medications or common problems like coughs, colds and rashes
  • If you want to talk to a professional but don't think it's an emergency call 111 or go to 111.nhs.uk

  • If you have any Covid-19 symptoms please ensure that you've been tested. If you test positive then contact everyone you've been incontact with i.e your workplace, school, family/friend's households, etc and isolate yourself.

  • When going to a testing station please ensure that you wear a mask and follow social distancing. If you want to learn more about social distancing find out more here
